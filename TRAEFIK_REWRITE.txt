AÇIKLAMA:
- rewrite-to-index middleware: Static dosyalar ve admin panel dışındaki tüm istekleri /index.php'ye yönlendirir
- Regex açıklaması:
  - ^/(?!assets|...) - assets ile başlamayan
  - |.*\.(jpg|...) - veya static dosya uzantısı olmayan
  - .*$ - tüm istekler
- Replacement: /index.php'ye yönlendir

ÖNEMLİ:
1. Tüm label'ları Coolify'deki Container Labels bölümüne ekleyin
2. Deploy sonrası clean URL'ler çalışmalı (örn: /about, /blog, /contact)
3. Static dosyalar (CSS, JS, images) direkt geçirilir
4. Admin panel (/hsnpanel2024secure/) direkt geçirilir
